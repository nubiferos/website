Welcome to NubiferOS! This section will guide you through downloading, installing, and configuring your secure cloud workstation.

## Prerequisites

Before you begin, ensure you have:

- A 64-bit x86 computer (AMD64/Intel 64)
- At least 4GB RAM (8GB recommended)
- 20GB free disk space
- A USB drive (8GB or larger) for installation
- Cloud provider credentials (AWS, Azure, or GCP)

## Installation Path

1. **[Installation](/docs/getting-started/installation/)** - Download the ISO and create a bootable USB
2. **[Quick Start](/docs/getting-started/quick-start/)** - Create your first workspace in 10 minutes
3. **[First Workspace](/docs/getting-started/first-workspace/)** - Deep dive into workspace configuration

## What You'll Learn

By the end of this guide, you'll be able to:

- Install NubiferOS with full disk encryption
- Create isolated workspaces for different cloud accounts
- Securely store and manage cloud credentials
- Switch between workspaces with visual context indicators
- Use read-only mode to prevent accidental changes
