# NubiferOS Downloads

ISO files will be available here when the alpha release is ready.

## Planned Files

- `nubiferos-alpha-1.0-amd64.iso` - Main installer ISO
- `SHA256SUMS` - Checksums for verification
- `SHA256SUMS.gpg` - GPG signature

## Building from Source

Until the ISO is available, you can build NubiferOS from source:

```bash
git clone https://github.com/nubiferos/nubiferos.git
cd nubiferos
./build.sh
```

See the [GitHub repository](https://github.com/nubiferos/nubiferos) for details.
